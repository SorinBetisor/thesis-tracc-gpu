# thesis-tracc-gpu

Thesis workspace for BSc work on **GPU-accelerated track reconstruction** with ACTS/traccc on Nikhef’s Stoomboot cluster.

This repo stores **documentation, scripts, and results** only. All actual code (traccc, ACTS, builds) lives on the cluster—e.g. `/data/alice/sbetisor/traccc`—and is developed there via Remote-SSH. See `docs/setup/` for infrastructure validation (P1) and CPU baseline (P2); `docs/important-links.md` for Nikhef KB and ACTS/traccc references.
