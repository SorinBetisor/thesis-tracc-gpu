# Important links

- **[CT User Documentation](https://kb.nikhef.nl/ct/)** — Nikhef knowledge base: accounts, storage, computing (Stoomboot, batch jobs, interactive nodes), software, security, and general FAQ.

- **[ACTS documentation](https://acts.readthedocs.io/en/latest/index.html)** — ACTS (A Common Tracking Software): experiment-independent toolkit for charged-particle track reconstruction in HEP, including geometry, propagation, fitting, seeding, and vertexing.

- **[Directory $HOME](https://kb.nikhef.nl/ct/Directory_home.html)** — Nikhef documentation on the home directory: intended use, quota, what to store there vs. `/data` or `/project`, and backup behaviour.

- **[Interactive CPU nodes](https://kb.nikhef.nl/ct/Interactive_CPU_nodes.html)** — How to access and use the Stoomboot interactive CPU nodes (stbc-i1/i2/i3) for analysis, scripting, and compiling; usage guidelines and limits.

- **[Interactive GPU nodes](https://kb.nikhef.nl/ct/Interactive_GPU_nodes.html)** — How to access and use the Stoomboot interactive GPU nodes (e.g. wn-lot-001), CUDA versions, and Python/conda setup for GPU work.

- **[traccc (GitHub)](https://github.com/acts-project/traccc)** — Demonstrator tracking chain on accelerators: clusterization, seeding, track finding, and fitting on CPU, CUDA, SYCL, and Alpaka.

- **[ACTS (GitHub)](https://github.com/acts-project/acts)** — ACTS source repository: experiment-independent tracking toolkit in modern C++; core library, plugins, Fatras, examples, and Python bindings.
